# Lita::Sushiyuki

Give me sushi

## Installation

Add this line to your lita's Gemfile:

    gem 'lita-sushiyuki'

And then execute:

    $ bundle

## Usage

```
Lita: sushi me ikura
Lita: sushi ikura
Lita: sushi ok
Lita: sushi list
...
```
